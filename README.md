# Topleo Bootstrap

Infrastructure as Code para todas as máquinas da Topleo.

Este projeto instala automaticamente um ambiente completo para:

- Bluefin
- Fedora
- Ubuntu
- Contabo VPS
- Payload CMS
- Ghost CMS
- Node.js
- IA
- Docker/Podman
- Desenvolvimento Web
- Rádio Digital
- Automações
